# Stack Overflow Developer Survey Analysis

## 📌 Project Overview

This project analyzes data from the Stack Overflow Developer Survey using Python and Pandas.

The goal of the analysis is to explore trends in:
- Python usage
- Work formats (remote / hybrid / in-office)
- Developer salaries
- Learning methods

---

## 📊 Key Analysis

The project includes:

- Data loading and preprocessing
- Handling missing values
- Statistical analysis (mean, median, mode)
- Filtering and grouping data
- Visualization using Matplotlib

---

## 📈 Key Insights

- Python is widely used among developers across different age groups
- Remote work is the most common work format
- Salary levels vary significantly between countries

---

## 🛠 Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

The dataset is based on the Stack Overflow Developer Survey.
